# iobroker-Dashboards-Scripts
Dashboards for iobroker without Grafana

EnergyCharts:
- Initial commit of the Energy-Charts script for ioBroker, version 1.5.0.
- This script includes functionality for fetching and displaying public power data, along with various helper functions and state management.

Synology Photos:
- This script show your photos on the synology with statistics and random found pictures in different states for vis/vis2/jarvis/lovelace

De Longhi: 
- This script read the statistics from your Eletta DeLonghi coffee machine and create a dashboard for vis/vis2/jarvis/lovelace/minuvis etc.
- needs the credentials from the coffee link app
